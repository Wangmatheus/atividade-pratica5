# Atividade Prática V — Desenvolvimento Web I

Projeto estático simulando uma rede social no estilo Twitter, desenvolvido com HTML e Pico CSS.

## Páginas

- **index.html** — Login
- **cadastro.html** — Cadastro de novo usuário
- **feed.html** — Feed de postagens

## Tecnologias

- HTML5
- Pico CSS v2

## Como executar

Basta abrir o arquivo `index.html` no navegador. A navegação entre as páginas funciona pelos links e botões internos.
